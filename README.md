# Eco-Track

## Description
Eco-Track is a smart eco-tracking application designed to estimate daily and weekly CO₂ emissions.
The project aims to create awareness about environmental impact using simple data analysis logic.

## Tech Stack
- TypeScript
- JavaScript

## Features
- Tracks CO₂ emissions over time
- Weekly and daily emission estimates
- Simple and clean logic implementation

## Learning Outcomes
- Logical problem-solving using TypeScript
- Data handling and calculation techniques
- Writing clean and maintainable code
